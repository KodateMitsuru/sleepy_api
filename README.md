# sleepy_api
sleepy api only version

| 路径                                   | 作用                |
| -------------------------------------- | ------------------- |
| `/query`                               | 获取状态            |
| `/get/status_list`                     | 获取可用状态列表    |
| `/set?secret=<secret>&status=<status>` | 设置状态 (url 参数) |
